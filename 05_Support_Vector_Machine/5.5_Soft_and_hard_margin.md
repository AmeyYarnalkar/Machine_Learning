In **soft margin SVM** (from Support Vector Machine), the constraint becomes:

$$
y_i (w^T x_i + c) \ge 1 - \xi_i
$$

where

* $y_i \in {-1, +1}$
* $\xi_i \ge 0$ (slack variable)

Now let's understand the **ranges of $y_i(w^Tx_i + c)$**.

---

## Case 1: Correctly classified and outside the margin

$$
y_i(w^Tx_i + c) \ge 1
$$

Meaning:

* point is correctly classified
* point lies **outside the margin**

These points **do not contribute to loss**.

---

## Case 2: Correctly classified but inside margin

$$
0 < y_i(w^Tx_i + c) < 1
$$

Meaning:

* still correctly classified
* but lies **inside the margin region**

These points produce **some hinge loss**.

---

## Case 3: Misclassified

$$
y_i(w^Tx_i + c) \le 0
$$

Meaning:

* classifier predicts the **wrong class**

These points produce **large hinge loss**.

---

## So the full ranges are

| Range of $y_i(w^Tx_i+c)$ | Meaning                   |
| ------------------------ | ------------------------- |
| ≥ 1                      | correct & outside margin  |
| 0 to 1                   | correct but inside margin |
| ≤ 0                      | misclassified             |
