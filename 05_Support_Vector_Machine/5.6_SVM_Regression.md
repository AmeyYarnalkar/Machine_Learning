
## 1. Regression model

In regression we predict a **continuous value**, so the model is

$$
f(x) = w^T x + c
$$

Notice one correction:
For regression the function is **not set equal to 0**. That was only for classification boundaries.

Here it is simply the **prediction line**.

---

## 2. ε-insensitive tube

SVR introduces an **epsilon tube** around the regression line.

Upper boundary:

$$
w^T x + c + \epsilon
$$

Lower boundary:

$$
w^T x + c - \epsilon
$$

Graphically:

```
upper tube       f(x)+ε
--------------------------

regression line  f(x)

--------------------------
lower tube       f(x)-ε
```

So points inside this tube are considered **good enough predictions**.

---

## 3. Core constraint

For each point $ (x_i, y_i) $:

$$
|y_i - (w^T x_i + c)| \le \epsilon
$$

Meaning:

Prediction error must stay within **epsilon tolerance**.

---

## 4. Allow violations (soft margin idea)

Real data won't perfectly fit inside the tube.

So we introduce slack variables:

* $ \xi_i $ → violation above tube
* $ \xi_i^* $ → violation below tube

Constraints become:

Upper violation:

$$
y_i - (w^T x_i + c) \le \epsilon + \xi_i
$$

Lower violation:

$$
(w^T x_i + c) - y_i \le \epsilon + \xi_i^*
$$

Where

$$
\xi_i, \xi_i^* \ge 0
$$

---

## 5. Optimization objective

Now we minimize two things:

1️. Model complexity \
2️. Error outside the tube

So the objective becomes

$$
\min \frac{1}{2}||w||^2 + C \sum_{i=1}^{n}(\xi_i + \xi_i^*)
$$

Where:

* $||w||^2$ → keeps the function **flat**
* $C$ → penalty for violating epsilon tube

---

# 6. Final intuition

SVR basically says:

> "I don't care about small errors."

If prediction error is within **ε**, loss = 0.

Only when points go **outside the ε-tube** do we start penalizing them.

This is called **ε-insensitive loss**.
