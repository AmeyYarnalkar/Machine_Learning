# Ensemble Techniques

## 1. What is Ensemble Learning?

Ensemble learning is a machine learning technique where **multiple models are combined together to make better predictions**.

Instead of depending on **one model**, we use **many models**, and their predictions are combined.

The main idea is:

> **Many weak or average models together can produce a strong and accurate model.**

This improves:

* Accuracy
* Stability
* Generalization

---

# Types of Ensemble Techniques

The two main types are:

1. **Bagging**
2. **Boosting**

---

# 1. Bagging (Bootstrap Aggregating)

## Idea

In bagging, we **split the dataset into multiple samples** and train **multiple models in parallel**.

Each model is called a **base learner**.

These models learn **independently** and **at the same time (parallel training)**.

---

## Step-by-Step Process

1. We start with a **dataset**.

2. The dataset is **divided into multiple samples**.

3. Each sample is given to a **different base learner**.

4. Each base learner trains **independently**.

5. These models can be:

   * Logistic Regression
   * Linear Regression
   * SVM
   * Decision Trees
   * Any other algorithm

6. Each model gets:

   * Some **samples of rows**
   * Some **samples of features**

7. Because each model sees a different part of the data, **each model becomes expert in its own portion**.

---

## Prediction Stage

When test data is given:

Each model produces a **prediction**.

Then we combine the predictions.

### For Classification

We take **majority voting**.

Example:

Model outputs →
A, A, B, A, B

Final prediction → **A**

---

### For Regression

We take the **average of predictions**.

Example:

Predictions →
10, 12, 11, 13

Final prediction →

[
\frac{10 + 12 + 11 + 13}{4} = 11.5
]

---

## Key Characteristics

* Training is **parallel**
* Models are **independent**
* Reduces **variance**
* Improves **stability**

---

# 2. Boosting

## Idea

Boosting is another ensemble technique where **models are trained sequentially**.

Here we use **weak learners**.

A **weak learner** is a model that performs only **slightly better than random guessing**.

Boosting converts **many weak learners into one strong learner**.

---

## Step-by-Step Process

1. Start with the **dataset**.

2. The dataset is given to the **first weak learner**.

3. The model trains and makes predictions.

4. Some records are **predicted correctly**, while some are **predicted poorly**.

5. The **records where the model performed poorly** are passed to the **next weak learner**.

6. The next weak learner focuses **more on these difficult records**.

7. This process continues **sequentially across many weak learners**.

8. After training multiple weak learners, we combine them to create a **strong learner**.

---

## Prediction Stage

All weak learners produce predictions.

These predictions are combined.

### For Classification

We take **majority voting** of all weak learners.

---

### For Regression

We take the **average of predictions**.

---

## Key Characteristics

* Training is **sequential**
* Models **depend on previous models**
* Focuses on **correcting previous mistakes**
* Converts **weak learners into strong learners**

---

# Quick Comparison

| Feature          | Bagging         | Boosting                    |
| ---------------- | --------------- | --------------------------- |
| Training         | Parallel        | Sequential                  |
| Model Dependency | Independent     | Dependent                   |
| Focus            | Reduce variance | Reduce bias                 |
| Learners         | Base learners   | Weak learners               |
| Example          | Random Forest   | AdaBoost, Gradient Boosting |
