Variants of **Naive Bayes** mainly differ in **how they model the feature distribution** $P(x_i|y)$. \
Different types of data → different variant.

---

## 1. Gaussian Naive Bayes

Used when **features are continuous numbers**.

Assumes each feature follows a **Gaussian Distribution**.


### Example use

* height
* weight
* temperature
* sensor values

### Real example

Predict **disease** based on:

* blood pressure
* cholesterol
* glucose level

---

## 2. Multinomial Naive Bayes

Used for **count-based features**.

Very common in **text classification**.

### Example features

Number of times words appear:

| word  | count |
| ----- | ----- |
| free  | 3     |
| offer | 2     |
| money | 1     |

### Applications

* spam detection
* document classification
* sentiment analysis

---

## 3. Bernoulli Naive Bayes

Used when features are **binary (0 or 1)**.

Instead of counts, it checks **presence or absence**.

Example:

| word  | present |
| ----- | ------- |
| free  | 1       |
| offer | 0       |
| money | 1       |

### Difference from multinomial

Multinomial → **counts frequency**
Bernoulli → **only presence/absence**

---

## 4. Complement Naive Bayes

Improved version of multinomial for **imbalanced datasets**.

Used a lot for **large text datasets**.

Handles situations where:

* some classes have **very few samples**

---

## 5. Categorical Naive Bayes

Used when features are **categorical variables**.

Example features:

* color: red / blue / green
* size: small / medium / large

Common in structured datasets.

---

# Quick Comparison

| Variant        | Data Type            | Example                 |
| -------------- | -------------------- | ----------------------- |
| Gaussian NB    | Continuous numbers   | medical data            |
| Multinomial NB | Counts               | word frequency          |
| Bernoulli NB   | Binary               | word present/not        |
| Complement NB  | Imbalanced text      | document classification |
| Categorical NB | categorical features | survey data             |
