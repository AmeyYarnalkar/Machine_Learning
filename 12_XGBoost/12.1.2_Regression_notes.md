# XGBoost Regressor – Working Notes

Using **XGBoost**

---

## Step 1 — Initial Prediction (Base Model)

For regression, the initial prediction is:

$$
Initial\ Prediction = Mean(y)
$$

Where:

* **y** = target values

This mean value becomes the **starting prediction for all samples**.

---

## Step 2 — Calculate Residuals

Residual represents **error in prediction**.

Formula:

$$
Residual = Actual - Prediction
$$

These residuals represent **how much the model is wrong**.

The next tree will try to **predict these residuals**.

---

## Step 3 — Build Decision Tree

A **decision tree is trained using residuals**.

Just like classifier:

❌ No entropy
❌ No gini

Instead **similarity weight** is used to determine the best split.

---

## Step 4 — Similarity Weight (Regression)

Formula:

$$
Similarity =
\frac{(\sum Residual)^2}
{n + \lambda}
$$

Where:

* **Residual** = error values
* **n** = number of residuals in the node
* **λ (lambda)** = regularization parameter

Difference from classifier:

```
Classifier cover → Σ P(1-P)
Regressor cover → number of samples (n)
```

---

## Step 5 — Gain Calculation

Gain determines the **best feature split**.

Formula:

$$
Gain =
Similarity(left)
+
Similarity(right)
-

Similarity(root)
$$

The feature with **highest gain** is chosen.

---

## Step 6 — Leaf Node Values

Each leaf node stores the **residual values of samples reaching that leaf**.

Prediction from the leaf is calculated as:

```
Average of residuals in that leaf
```

---

## Step 7 — Prediction from Tree

For a record:

1. Pass features through the tree
2. Reach a leaf node
3. Take **average residual of that leaf**

---

## Step 8 — Update Prediction

Prediction is updated using learning rate.

$$
New\ Prediction =
Old\ Prediction +
(\eta \times Leaf\ Output)
$$

Where:

* **η** = learning rate

---

## Final Prediction Pipeline

```
Initial Prediction = Mean(y)
        ↓
Compute Residuals
        ↓
Train Decision Tree on Residuals
        ↓
Choose Best Split using Gain
        ↓
Leaf stores residual values
        ↓
Leaf output = average residual
        ↓
Update prediction with learning rate
```

---

## Key Difference from Classifier

| Concept            | XGBoost Classifier  | XGBoost Regressor         |
| ------------------ | ------------------- | ------------------------- |
| Initial Prediction | Log-odds            | Mean of target            |
| Cover Value        | Σ P(1-P)            | Number of samples         |
| Leaf Output        | Similarity weight   | Average residual          |
| Final Step         | Sigmoid probability | Direct numeric prediction |

---

One small **important correction** (conceptually):

In regression trees **XGBoost technically stores the leaf weight**, not raw residuals — but that weight is mathematically equivalent to **mean residual adjustment**, which is why our understanding works for intuition.
