# XGBoost Classifier 

## Step 1 — Initial Prediction (Base Model)

XGBoost starts with a **base prediction using log-odds**.

Formula:

$$
\text{log odds} = \log\left(\frac{P}{1-P}\right)
$$

Where:

* **P** = probability of class **1**

This value becomes the **initial prediction for all samples**.

---

## Step 2 — Calculate Residuals

Residual represents **prediction error**.

Formula:

$$
\text{Residual} = \text{Actual} - \text{Prediction}
$$

Residual tells the model:

* how far prediction is from the true value

These residuals are used to **train the next tree**.

---

## Step 3 — Build Decision Tree

A **decision tree is trained on residuals**.

But unlike normal decision trees:

❌ **No Entropy** \
❌ **No Gini Index**

Instead XGBoost uses **Similarity Weight**.

---

## Step 4 — Similarity Weight

Similarity weight measures **how pure a node is**.

Formula:

$$
Similarity\ Weight =
\frac{(\sum Residual)^2}
{\sum (P(1-P)) + \lambda}
$$

Where:

* **Residual** = error for each sample
* **P** = probability prediction of each sample
* **λ (lambda)** = regularization parameter

Purpose of λ:

* prevents overfitting
* penalizes large leaf weights

---

## Step 5 — Gain Calculation (Best Split)

Gain measures **how good a split is**.

Formula:

$$
Gain =
Similarity(left)
+
Similarity(right)
-

Similarity(root)
$$

Interpretation:

* calculate similarity of left child
* calculate similarity of right child
* subtract similarity of parent node

The **feature with highest gain** is selected.

---

## Step 6 — Leaf Output (Prediction Value)

In regular trees:

```
leaf output = average value
```

In XGBoost:

```
leaf output = similarity weight
```

Each leaf stores a **weight value**.

---

## Step 7 — Prediction from Tree

For each record:

1. Pass features through the tree
2. Reach a leaf node
3. Take **leaf weight as prediction update**

---

## Step 8 — Update Model Prediction

Update rule:

$$
New\ Prediction =
Old\ Prediction +
(\eta \times Leaf\ Weight)
$$

Where:

* **η (learning rate)** controls update size.

---

## Step 9 — Convert to Probability

Final step uses **sigmoid function**.

$$
Probability =
\frac{1}{1 + e^{-prediction}}
$$

This converts log-odds into **probability of class 1**.

---

## Final Prediction Pipeline

```
Initial Log-Odds Prediction
        ↓
Compute Residuals
        ↓
Train Decision Tree
        ↓
Find Best Split using Gain
        ↓
Store Similarity Weight in Leaves
        ↓
Update Prediction using Learning Rate
        ↓
Apply Sigmoid
        ↓
Final Probability
```
