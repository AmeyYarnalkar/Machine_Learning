# Ordinary Least Squares (OLS)

## Definition

```id="ols_def1"
OLS is a method used to find the best fit line
by minimizing the squared errors between
actual and predicted values
```

---

## Core Idea

```id="ols_def2"
Find parameters (β0, β1) such that
total squared error is minimum
```

---

## Model

```id="ols_def3"
ŷ = β0 + β1x
```

---

## Objective

```id="ols_def4"
Minimize:

Σ (yi - ŷi)^2
```

---

## Why “Least Squares”?

```id="ols_def5"
"Least" → minimize error
"Squares" → square the errors
```

---

## Why Squared Error?

```id="ols_def6"
• Makes error always positive
• Penalizes large errors more
• Gives smooth curve → easy optimization
• Ensures one global minimum
```

---

## What OLS Does

```id="ols_def7"
• Takes all data points
• Calculates prediction error
• Squares the error
• Adds them up
• Finds parameters that minimize this sum
```

---

## Output of OLS

```id="ols_def8"
β0 → intercept
β1 → slope
```

These define the **best fit line**.

## Final Relationship

### Final Relationship (OLS Linear Regression)

```id="rel1"
ŷ = β0 + β1x
```

This is the **final best fit line** obtained using OLS.

---

### Expanded Form (Data Points)

```id="rel2"
ŷi = β0 + β1xi
```

For each data point `i`.

---

### Final Parameter Formulas

```id="rel3"
β1 = Σ (xi - x̄)(yi - ȳ) / Σ (xi - x̄)^2

β0 = ȳ - β1 x̄
```

---

### Full Pipeline Relationship

```id="rel4"
Data → Error → Squared Error → Sum → Minimize → Best β0, β1 → Final Line
```

---

## Notation Meaning

### Variables

```id="rel5"
xi → input value
yi → actual output
ŷi → predicted output
```

---

### Parameters

```id="rel6"
β0 → intercept (value of y when x = 0)

β1 → slope (change in y per unit change in x)
```

---

### Means

```id="rel7"
x̄ → mean of all x values
ȳ → mean of all y values
```

---

### Summation

```id="rel8"
Σ → sum over all data points
```

---

### Error

```id="rel9"
error = yi - ŷi
```

---

### Cost Function

```id="rel10"
J = (1/n) * Σ (yi - ŷi)^2
```

---

### Key Interpretation

```id="rel11"
β1 → strength and direction of relationship

β0 → baseline value of y

ŷ → model prediction

J → how wrong the model is overall
```
