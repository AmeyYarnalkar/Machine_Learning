# AdaBoost Algorithm

## Step 1 — Creating the First Decision Stump

The first step in the AdaBoost algorithm is to **create the first weak learner**, which is typically a **decision stump**.

A **decision stump** is a decision tree with only **one split**.

This means:

* maximum depth = 1
* one decision rule
* two leaf nodes

Because the model is extremely simple, it acts as a **weak learner**.

---

## How the First Decision Stump is Created

To construct the first stump, the algorithm evaluates **every feature** in the dataset.

For each feature:

1. The algorithm considers possible **split points**.
2. A stump is built using that feature and split.
3. The quality of the split is evaluated.
4. The best split is selected.

The evaluation metric depends on the **type of problem**.

---

## Case 1 — Classification

For classification problems, the split quality is evaluated using **Information Gain**.

Information Gain is computed using **Entropy**.

Entropy (information theory)

$$H(S)=-\sum_{i=1}^{c} p_i \log_2 p_i$$

Where:

* (p_i) = probability of class (i)
* (c) = number of classes

Entropy measures **impurity in the dataset**.

Lower entropy means **better class separation**.

---

### Information Gain

Information Gain

$$IG(S,A)=H(S)-\sum_{v \in Values(A)} \frac{|S_v|}{|S|}H(S_v)$$

Where:

* (S) = dataset
* (A) = feature
* (S_v) = subset after split

The feature producing the **highest Information Gain** is selected.

That feature becomes the **decision stump split**.

---

## Case 2 — Regression

For regression problems, the algorithm evaluates splits using **Variance Reduction**.

Variance

$$Var(S)=\frac{1}{n}\sum_{i=1}^{n}(y_i-\bar{y})^2$$

Variance measures how spread out the target values are.

Lower variance means **better grouping of similar values**.

---

### Variance Reduction

Variance reduction

$$VR = Var(S)-\sum_{i=1}^{k}\frac{|S_i|}{|S|}Var(S_i)$$

Where:

* $S$ = original dataset
* $S_i$ = subsets after split

The split with the **maximum variance reduction** is selected.

---

## Final Selection of the First Stump

After evaluating all possible splits across all features:

* The split with **maximum information gain (classification)**
* or **maximum variance reduction (regression)**

is selected.

This becomes the **first decision stump (weak learner)** of AdaBoost.
