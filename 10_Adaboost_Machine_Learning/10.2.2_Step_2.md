
# Step 2 

## Initialize Sample Weights

Before training the first weak learner, AdaBoost assigns a **weight to every training sample**.

Initially, **all samples are equally important**.

If the dataset contains **N records**, then each record receives the weight:

$$
w_i = \frac{1}{N}
$$

Where:

* $w_i$ = weight of sample $i$
* $N$ = total number of samples

### Important Property

The weights are normalized so that:

$$
\sum_{i=1}^{N} w_i = 1
$$

This means the **total importance of all samples equals 1**.

---

## Pass Data Through First Weak Learner

The dataset with these weights is then passed to the **first decision stump**.

The stump predicts the class for every record.

After prediction, the samples fall into two groups:

* **Correctly classified samples**
* **Misclassified samples**

The misclassified samples are considered **errors**.

---

## Calculate Weighted Error

Instead of simply counting mistakes, AdaBoost calculates the **weighted error**.

AdaBoost weighted error

$$
error = \sum w_i
$$

Where the summation is taken **only over misclassified samples**.

So effectively:

```
Weighted Error =
Sum of weights of misclassified samples
```

This is important because **later iterations will change the sample weights**, so some errors become more important than others.

---

## Compute Stump Performance (Model Weight)

Now we compute how good this stump is.

This value is called **alpha** (or lambda in some notes).

AdaBoost model weight

$$
\alpha = \frac{1}{2} \ln\left(\frac{1-error}{error}\right)
$$

Where:

* $error$ = weighted error of the stump
* $\alpha$ = weight assigned to the model

This value determines **how much influence the stump will have in the final prediction**.

---

## Interpretation of Alpha (Very Important)

If the error is **small**

```
error → 0
```

Then

```
alpha → large
```

Meaning:

**The model is strong and receives high importance.**

---

If the error is **close to 0.5**

```
error ≈ 0.5
```

Then

```
alpha ≈ 0
```

Meaning:

**The model has almost no predictive power.**
