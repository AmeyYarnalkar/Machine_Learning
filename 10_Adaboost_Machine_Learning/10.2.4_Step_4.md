

# Step 4 

## Sampling Records for the Next Weak Learner

After updating the sample weights, the next weak learner must be trained.

The goal is to ensure that **misclassified samples from the previous model have a higher probability of being selected for training**.

To achieve this, we construct **weighted sampling bins**.

---

## Creating Weight Bins

For every record, we create a **bin (range)** based on its weight.

Let the weights after update be:

```
w1, w2, w3, ... , wN
```

Since weights are normalized:

```
w1 + w2 + ... + wN = 1
```

We construct cumulative bins.

Example:

| Record | Weight | Bin Range              |
| ------ | ------ | ---------------------- |
| R1     | w1     | 0 → w1                 |
| R2     | w2     | w1 → w1 + w2           |
| R3     | w3     | w1 + w2 → w1 + w2 + w3 |
| ...    | ...    | ...                    |
| RN     | wN     | previous → 1           |

This creates a **cumulative probability distribution**.

---

## Why Misclassified Samples Get Larger Bins

In the previous step:

* **Misclassified samples had their weights increased**
* **Correct samples had their weights decreased**

Therefore:

* misclassified samples → **larger bin ranges**
* correctly classified samples → **smaller bin ranges**

So they occupy **more space in the probability distribution**.

---

## Selecting Records for Next Training

Now we perform **random sampling**.

Procedure:

1. Generate a **random number between 0 and 1**.
2. Find the **bin range that contains that number**.
3. Select the corresponding record.

Example:

```
Random number generated = 0.63
```

Suppose bins are:

```
R1 → 0 – 0.10
R2 → 0.10 – 0.25
R3 → 0.25 – 0.60
R4 → 0.60 – 0.80
R5 → 0.80 – 1.00
```

Since **0.63 lies in R4**, record **R4 is selected**.

---

## Key Insight

Because misclassified samples have **larger bins**, they have a **higher probability of being selected multiple times**.

This forces the next weak learner to **focus on difficult samples**.

---

## Intuition (Very Important)

Think of the interval **0 → 1** as a **roulette wheel**.

Each sample owns a portion of the wheel proportional to its weight.

Bigger weight → bigger slice → higher chance of selection.

---

## Final Step

After sampling **N records**, the next **decision stump is trained**.

This process repeats for **M weak learners**.

Finally all models are combined:

$$
F(x) = \sum_{m=1}^{M} \alpha_m h_m(x)
$$
