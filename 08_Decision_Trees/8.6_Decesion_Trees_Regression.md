# Decision Tree Regression

### What is the Difference from Classification?

In **decision tree classification**, the target variable is **categorical**.

Examples:

* Spam / Not Spam
* Fraud / Legitimate
* Yes / No

But in **decision tree regression**, the target variable is **continuous**.

Examples:

* House price prediction
* Temperature prediction
* Sales forecasting

Instead of predicting a **class label**, the model predicts a **numerical value**.

Regression trees are commonly implemented using the CART algorithm.

---

## How a Decision Tree is Constructed for Regression

The tree construction process is similar to classification, but the **splitting criterion is different**.

Instead of impurity measures like entropy or Gini, regression trees use **variance reduction**.

---

## Handling Different Types of Features

### 1. Categorical Features

If the feature is **categorical**, each category can be considered as a possible split.

Example:

```text
Feature: Color
Values: Red, Blue, Green
```

The tree evaluates splits such as:

```
Color = Red ?
Color = Blue ?
Color = Green ?
```

Each split divides the dataset and evaluates how good the split is.

---

### 2. Numerical Features

If the feature is **numerical**, the algorithm first **sorts the feature values**.

Example:

```text
Feature values: 10, 15, 20, 30
```

Possible split points are generated between consecutive values:

```
x ≤ 12.5
x ≤ 17.5
x ≤ 25
```

For each possible threshold, the algorithm evaluates the quality of the split.

---

## Choosing the Best Feature (Variance Reduction)

In regression trees, the best split is chosen using **variance reduction**.

The idea is simple:

A good split should **reduce the variance of the target values**.

If the values within a node are very similar, the variance becomes **small**, meaning the node is more pure.

---

## Variance Formula

$$Var(S) = \frac{1}{n} \sum_{i=1}^{n}(y_i - \bar{y})^2$$

### Meaning

* **S** → dataset at the node
* **n** → number of samples
* **yᵢ** → target value
* **ȳ** → mean of the target values

Variance measures **how spread out the target values are**.

---

## Variance Reduction Formula

$$VR = Var(parent) - (w_L Var(left) + w_R Var(right))$$

---

## Formula Meaning

**Var(parent)** :
Variance of the target values in the current node.

**Var(left)** :
Variance of the left subset after splitting.

**Var(right)**:
Variance of the right subset after splitting.

**w_L , w_R**:
Weights representing the proportion of samples.

$
w_L = \frac{|S_L|}{|S|}
$

$
w_R = \frac{|S_R|}{|S|}
$

These weights represent the **subset size relative to the entire dataset**.

---

## Selecting the Best Feature

For each feature:

1. Generate possible splits.
2. Compute variance for:

   * root node
   * left subset
   * right subset.
3. Calculate **variance reduction**.
4. Compare results across all features.

The feature with the **highest variance reduction** is selected.

---

## Tree Construction Process

The algorithm repeats the following steps:

1. evaluate all possible splits
2. compute variance reduction
3. choose the best split
4. divide the dataset into subsets
5. repeat the same process for each child node

This continues until a stopping condition is reached.

---

## Prediction at the Leaf Node

When the algorithm reaches a **leaf node**, it must produce a prediction.

Since this is regression, the prediction is the **average of all target values in that node**.

Formula for prediction:

$\hat{y} = \frac{1}{n} \sum_{i=1}^{n} y_i$

So the leaf node output becomes the **mean value of the subset** of training samples present in that node.

---

## Key Idea

Decision tree regression works by:

1. splitting the dataset to **reduce variance**
2. creating nodes where target values are **as similar as possible**
3. predicting the **average value at leaf nodes**

This allows the model to approximate complex **non-linear relationships** in the data.
