
#  Regularization

## Ridge Regression (L2 Regularization)



### 1. Basic Linear Regression Loss Function

$$
\text{Loss} = \frac{1}{2m} \sum_{i=1}^{m} (y_i - h(x_i))^2
$$

* $ y_i $ → actual value
* $ h(x_i) $ → predicted value
* $ m $ → number of data points

👉 Goal: Minimize this loss using **Gradient Descent** \
👉 Result: Best-fit line (optimal intercept + slope)

---

### 2. Problem: Overfitting

* Model performs **too well on training data**
* But performs **poorly on test data**

👉 Reason:

* Model is too complex
* It captures **noise instead of pattern**

---

###  3. Core Intuition Behind Regularization

> If we use the same loss function → we reach the same global  minimum → overfitting persists

👉 So we **modify the loss function**

---

### 4. Ridge Regression Loss Function (L2)

$$
\text{Loss} = \frac{1}{2m} \sum_{i=1}^{m} (y_i - h(x_i))^2 + \lambda \sum_{j=1}^{n} b_j^2
$$

* $ \lambda $ → regularization parameter (hyperparameter)
* $ b_j $ → model coefficients (slopes)

---

### 5. What this modification does

* Adds a **penalty for large coefficients**
* Prevents model from becoming too complex

👉 Effect:

* Slopes become **smaller**
* Model becomes **smoother and more generalizable**

---

### 6. Important Insight

* Without regularization → model may reach **overfitting minimum**
* With regularization → we **shift the solution** to a simpler region

👉 We are NOT avoiding global minimum \
👉 We are **changing what “optimal” means**

---

### 7. Role of Lambda (λ)

| λ Value | Effect                                                  |
| ------- | ------------------------------------------------------- |
| λ = 0   | Same as Linear Regression (no regularization)           |
| Small λ | Slight penalty → balanced model                         |
| Large λ | Strong penalty → very small coefficients (may underfit) |

---

### 8. Hyperparameter Tuning

👉 Choosing the right λ is critical

* Too small → overfitting
* Too large → underfitting

✔ Done using:

* Cross-validation (We will learn in the Upcoming Lessons)

---

###  9. Key Characteristics of Ridge

* Shrinks coefficients
* **Never makes them exactly zero**
* Keeps all features
