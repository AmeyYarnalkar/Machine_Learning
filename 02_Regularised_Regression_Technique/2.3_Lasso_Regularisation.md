
# Lasso Regression (L1 Regularization)


### 1. Problem Recap

* Linear regression can **overfit**
* Reason → coefficients become **too large / too complex**

👉 Need to control model → **regularization**

---

### 2. Lasso Loss Function (L1)

$$
\text{Loss} = \frac{1}{2m} \sum_{i=1}^{m} (y_i - h(x_i))^2 + \lambda \sum_{j=1}^{n} |b_j|
$$


* $ \lambda $ → regularization strength
* $ b_j $ → coefficients

---

### 3. Core Idea

👉 Penalizes **absolute values of coefficients**

---

### 4. What Lasso does (VERY IMPORTANT)

* Shrinks coefficients
* **Some coefficients become EXACTLY 0**

👉 This is the key difference from Ridge

---

### Why this is powerful

👉 Lasso automatically does **feature selection**

* Important features → kept
* Useless features → removed (coefficient = 0)

---

###  5. Intuition (real talk)

Think like:

> “If a feature is not helping enough, just kill it.”

Ridge says:

> “Reduce impact”

Lasso says:

> “Remove it completely”

---

### 6. Role of Lambda (λ)

| λ Value | Effect                                               |
| ------- | ---------------------------------------------------- |
| λ = 0   | Normal Linear Regression                             |
| Small λ | Few coefficients shrink                              |
| Large λ | Many coefficients → 0 (aggressive feature selection) |

---

### 7. When to use Lasso

👉 Use when:

* Many features
* Some features are **irrelevant**
* You want **simpler model**

---

### Limitation

* If features are highly correlated:

  * Lasso may pick **one randomly**
  * Ignore others

---

## 🔹 8. Ridge vs Lasso (quick clarity)

| Feature           | Ridge (L2)  | Lasso (L1)                         |
| ----------------- | ----------- | ---------------------------------- |
| Coefficients      | Small       | Some become 0                      |
| Feature Selection |  No        |  Yes                              |
| Stability         | More stable | Less stable (with correlated data) |

---

### Final takeaway

* Lasso = **shrink + eliminate features**
* Helps reduce overfitting AND complexity
* Useful for **feature selection**
