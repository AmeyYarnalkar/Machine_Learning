

# Gradient Boosting Regression

## Definition (Very Simple)

**Gradient Boosting** is a machine learning algorithm that improves predictions **step by step**.

Instead of building one complex model, it builds **many small decision trees**, where each new tree learns to **fix the mistakes made by the previous model**.

So the prediction gradually becomes **more accurate over time**.

---

## Purpose

The goal of gradient boosting is to **reduce prediction error by repeatedly correcting the remaining mistakes**.

Each tree does not predict the final value directly.
Instead, it predicts **how much correction should be added to the current prediction**.

---

## Step 1 — Initialize the Model

First we make a very simple prediction.

We calculate the **mean of all target values**.

```
Initial prediction = mean(y)
```

This value is used as the **starting prediction for every record**.

---

## Step 2 — Calculate Residuals

Residuals represent **the error made by the current model**.

```
Residual = Actual value − Predicted value
```

These residuals tell us **how wrong our prediction is for each record**.

---

## Step 3 — Train a Decision Tree on Residuals

Now we train a **decision tree** using:

```
Input features → X
Target values → Residuals
```

So the tree is learning **patterns in the errors**.

In regression trees:

* each leaf node predicts the **average residual** of the samples inside that leaf.

---

## Step 4 — Predict Residuals Using the Tree

Now we pass the **same input features** through the tree.

The tree outputs **predicted residuals (corrections)**.

---

## Step 5 — Update the Model Prediction

We update the prediction by **adding the correction predicted by the tree**.

```
New Prediction = Old Prediction + Tree Prediction
```

This moves the prediction **closer to the true value**.

---

## Step 6 — Compute New Residuals

Now we compute **new residuals** using the updated predictions.

```
Residual = Actual value − Updated prediction
```

These residuals are **smaller than before**, because the model has improved.

---

## Step 7 — Repeat the Process

We repeat the same steps:

1. Train a tree on new residuals
2. Predict corrections
3. Update the model prediction

Each new tree **reduces the remaining error**.

---

## Final Model

The final prediction is the **sum of all corrections**.

```
Final Prediction =
Initial prediction
+ Tree1 prediction
+ Tree2 prediction
+ Tree3 prediction
+ ...
```

Each tree contributes a **small improvement** to the model.

---

## Important Idea

Gradient boosting works because it turns a difficult prediction problem into **many small error-correction steps**.

Each tree simply answers:

```
“How should we correct the current prediction?”
```

---

## Key Hyperparameters

Important parameters controlling the model:

* **learning_rate** → controls how big each correction step is
* **n_estimators** → number of trees added to the model
* **max_depth** → depth of each decision tree
* **min_samples_split / min_samples_leaf** → control tree structure

These parameters help **prevent overfitting**.
