

# Multiple Linear Regression

## Definition

Multiple Linear Regression is used when:

* There is **one output (dependent variable)**
* There are **multiple input features (independent variables)**

Example

```
Inputs
x1 → house size
x2 → number of bedrooms
x3 → house age

Output
y → house price
```

---

# Hypothesis Function (Model Equation)

The prediction equation becomes:

```
h(x) = θ0 + θ1x1 + θ2x2 + ... + θnxn
```

Where

```
θ0 → intercept (bias)
θ1 → weight for feature x1
θ2 → weight for feature x2
...
θn → weight for feature xn
```

Example

```
HousePrice = θ0 
           + θ1 * size
           + θ2 * bedrooms
           + θ3 * age
```

Each feature contributes **some amount to the final prediction**.

---

# Geometric Interpretation

In simple linear regression we fit a **line**.

```
2 variables → line
```

In multiple linear regression we fit a **plane or hyperplane**.

```
3 variables → plane
4+ variables → hyperplane (cannot visualize)
```

So the model tries to find the **best hyperplane that fits the data**.

---

# Cost Function

The cost function still measures **how far predictions are from actual values**.

Formula

```
J(θ) = (1 / 2m) * Σ (h(xi) - yi)^2
```

Where

```
m  → number of training examples
h(xi) → predicted value
yi → actual value
```

Prediction function

```
h(xi) = θ0 + θ1xi1 + θ2xi2 + ... + θnxin
```

So the cost function becomes

```
J(θ) = (1 / 2m) * Σ (θ0 + θ1xi1 + θ2xi2 + ... + θnxin - yi)^2
```

Goal:

```
minimize J(θ)
```

---

# Gradient Descent (Convergence Algorithm)

We update **all parameters**.

```
θ0
θ1
θ2
...
θn
```

General update rule

```
θj = θj - α * ( ∂J / ∂θj )
```

Where

```
j = 0,1,2,...,n
α = learning rate
```

---

# Derivative of Cost Function

The partial derivative becomes

```
∂J / ∂θj = (1 / m) * Σ (h(xi) - yi) * xij
```

Where

```
xij → value of feature j in training example i
```

For θ0 specifically

```
∂J / ∂θ0 = (1 / m) * Σ (h(xi) - yi)
```

Because

```
x0 = 1
```

---

# Final Gradient Descent Updates

Update intercept

```
θ0 = θ0 - α * (1/m) * Σ (h(xi) - yi)
```

Update weights

```
θ1 = θ1 - α * (1/m) * Σ (h(xi) - yi)xi1
θ2 = θ2 - α * (1/m) * Σ (h(xi) - yi)xi2
...
θn = θn - α * (1/m) * Σ (h(xi) - yi)xin
```

---

# Convergence in Multiple Linear Regression

Convergence means:

```
parameters stop changing significantly
```

or

```
cost function stops decreasing
```

During training:

```
J(θ) decreases
J(θ) decreases
J(θ) decreases
```

Eventually

```
J(θ) reaches minimum
```

At this point:

```
θ0, θ1, θ2 ... θn represent the best fit hyperplane
```

---

# Final Goal

Find parameters

```
θ0, θ1, θ2, ... θn
```

such that

```
J(θ) is minimized
```

This produces the **best prediction model**.
