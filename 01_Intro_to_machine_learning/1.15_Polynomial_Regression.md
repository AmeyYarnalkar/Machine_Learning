

# Polynomial Regression

## Definition

```id="pr1"
Polynomial Regression is used when the relationship
between input and output is not linear
```

---

# Why Polynomial Regression?

```id="pr2"
Linear regression ג†’ straight line

Real data ג†’ curved relationship

Polynomial regression ג†’ captures curves
```

---

# What is a Polynomial?

## Definition

```id="pr3"
A polynomial is a mathematical expression
made of variables raised to powers
```

Example

```id="pr4"
x, x^2, x^3, x^4 ...
```

---

# What is Degree?

## Definition

```id="pr5"
Degree = highest power of the variable
```

Examples

```id="pr6"
y = x        ג†’ degree 1 (linear)

y = x^2      ג†’ degree 2 (quadratic)

y = x^3      ג†’ degree 3 (cubic)
```

---

# Polynomial Representation (Single Feature)

## General Form

```id="pr7"
yּ‚ = ־²0 + ־²1x + ־²2x^2 + ־²3x^3 + ... + ־²nx^n
```

---

# How It Varies with Degree

## Degree 1 (Linear)

```id="pr8"
yּ‚ = ־²0 + ־²1x
```

---

## Degree 2 (Quadratic)

```id="pr9"
yּ‚ = ־²0 + ־²1x + ־²2x^2
```

---

## Degree 3 (Cubic)

```id="pr10"
yּ‚ = ־²0 + ־²1x + ־²2x^2 + ־²3x^3
```

---

## Key Idea

```id="pr11"
Higher degree ג†’ more flexible curve
Lower degree ג†’ simpler curve
```

---

# Important Insight

```id="pr12"
Polynomial regression is still linear regression
in terms of parameters (־²)
```

Because:

```id="pr13"
It is linear in ־², not in x
```

---

# Multiple Features Polynomial Regression

## Case: Multiple Inputs

```id="pr14"
x1, x2 ג†’ inputs
y ג†’ output
```

---

## Degree 1 (Linear)

```id="pr15"
yּ‚ = ־²0 + ־²1x1 + ־²2x2
```

---

## Degree 2 (Quadratic with interactions)

```id="pr16"
yּ‚ = ־²0 
   + ־²1x1 + ־²2x2
   + ־²3x1^2 + ־²4x2^2
   + ־²5x1x2
```

---

## Degree 3 (Cubic with interactions)

```id="pr17"
yּ‚ = ־²0
   + ־²1x1 + ־²2x2
   + ־²3x1^2 + ־²4x2^2 + ־²5x1x2
   + ־²6x1^3 + ־²7x2^3
   + ־²8x1^2x2 + ־²9x1x2^2
```

---

# Key Insight (Very Important)

```id="pr18"
As degree increases ג†’ number of terms increases rapidly
```

---

# What Are Interaction Terms?

```id="pr19"
x1x2 ג†’ interaction between features
```

Meaning:

```id="pr20"
Effect of one feature depends on another
```

---

# Behavior with Degree

```id="pr21"
Low degree ג†’ high bias (underfitting)

High degree ג†’ high variance (overfitting)
```

---

# Final Summary

```id="pr22"
Polynomial regression models non-linear relationships
by adding powers and interactions of features
while still using linear regression techniques
```

---

# One-Line Understanding

```id="pr23"
Polynomial regression = linear regression on transformed features
```
