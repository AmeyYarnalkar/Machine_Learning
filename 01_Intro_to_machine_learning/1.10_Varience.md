
# Variance

## Definition

```id="v21"
Variance = how much the model changes when the training data changes
```

---

## Core Intuition

Train the same model on two slightly different datasets:

```id="v22"
Dataset A → Model A (one curve)

Dataset B → Model B (different curve)
```

If the models are very different:

```id="v23"
High Variance
```

If they are almost the same:

```id="v24"
Low Variance
```

---

## What Variance Measures

```id="v25"
Sensitivity of the model to small changes in data
```

---

## Key Idea

```id="v26"
Small change in data → big change in model → High Variance

Small change in data → small change in model → Low Variance
```

---

## High Variance

## Meaning

```id="v27"
Model is unstable and overfits the data
```

---

## Behavior

```id="v28"
• Fits training data very closely
• Captures noise along with pattern
• Performs poorly on new (test) data
```

---

## Interpretation

```id="v29"
High Variance → Overfitting
```

---

## Low Variance

## Meaning

```id="v30"
Model is stable and consistent
```

---

## Behavior

```id="v31"
• Predictions do not change much with new data
• Generalizes better
```

---

## Sources of Variance (Very Important)

## 1. Model Complexity

```id="v32"
Highly complex model → High Variance
Simple model → Low Variance
```

Example:

```id="v33"
Deep neural network → High variance
Linear regression → Low variance
```

---

## 2. Small Dataset

```id="v34"
Less data → model depends heavily on few points
→ High Variance
```

---

## 3. Noise in Data

```id="v35"
Random fluctuations in data
→ model tries to fit noise
→ High Variance
```

---

## 4. Outliers

```id="v36"
Extreme values pull the model
→ large changes in learned pattern
```

---

## 5. High Dimensional Data (Many Features)

```id="v37"
Too many features → model becomes flexible
→ High Variance
```

---

## 6. Poor Data Distribution

```id="v38"
Training data not representative
→ model changes drastically with new data
```

---

## Visual Intuition

```id="v39"
High Variance → very wiggly curve (fits every point)

Low Variance → smooth curve (captures general trend)
```

---

# Important Clarification

```id="v40"
Variance is NOT about difference between datasets
Variance is about difference in model behavior due to datasets
```

---

## Final Summary

```id="v41"
Variance = how much your model overreacts to data
```

---

## One-Line Understanding

```id="v42"
Variance = instability of the model when training data changes
```
