
# Overfitting and Underfitting

## Data Split

```id="of1"
Dataset
   → Training Dataset
        → Train Set
        → Validation Set
   → Test Dataset
```

---

## Overfitting

### Definition

```id="of2"
Model performs very well on training data
but poorly on test data
```

---

### Behavior

```id="of3"
Training Accuracy → High
Test Accuracy     → Low
```

---

### Reason

```id="of4"
Model learns noise and memorizes data
instead of learning real patterns
```

---

### Bias-Variance

```id="of5"
Low Bias
High Variance
```

---

## Underfitting

### Definition

```id="of6"
Model performs poorly on both
training and test data
```

---

### Behavior

```id="of7"
Training Accuracy → Low
Test Accuracy     → Low
```

---

### Reason

```id="of8"
Model is too simple to capture patterns
```

---

##$ Bias-Variance

```id="of9"
High Bias
Low Variance (usually)
```

---

## Goal

```id="of10"
Low Bias + Low Variance
```
