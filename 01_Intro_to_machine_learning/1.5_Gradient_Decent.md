Below is the **derivation of Gradient Descent for Linear Regression** in a clean step-by-step way.

# Given

```

```

# Step 1 ‚ÄĒ Substitute the hypothesis

Replace `hőł(x)` in the cost function.

```
J(őł0, őł1) = (1 / 2m) * ő£ (őł0 + őł1xi - yi)^2
```

# Step 2 ‚ÄĒ Derivative w.r.t őł0

We compute

```
‚ąāJ / ‚ąāőł0
```

Derivative of squared term

```
d/dőł0 (őł0 + őł1xi - yi)^2
```

Substitute into cost function derivative.

```
‚ąāJ/‚ąāőł0 = (1 / 2m) * ő£ 2(őł0 + őł1xi - yi)
```

Cancel `2`.

```
‚ąāJ/‚ąāőł0 = (1 / m) * ő£ (őł0 + őł1xi - yi)
```

Since

```
hőł(xi) = őł0 + őł1xi
```

```
‚ąāJ/‚ąāőł0 = (1 / m) * ő£ (hőł(xi) - yi)
```

# Step 3 ‚ÄĒ Derivative w.r.t őł1

Now compute

```
‚ąāJ / ‚ąāőł1
```

Derivative of squared term


Substitute into cost function derivative.

```
‚ąāJ/‚ąāőł1 = (1 / 2m) * ő£ 2(őł0 + őł1xi - yi)xi
```

```
‚ąāJ/‚ąāőł1 = (1 / m) * ő£ (őł0 + őł1xi - yi)xi
```

Replace hypothesis.

Final result

```
‚ąāJ/‚ąāőł1 = (1 / m) * ő£ (hőł(xi) - yi)xi
```

---

# Step 4 ‚ÄĒ Gradient Descent Update Rules

Now plug derivatives into gradient descent.

Update őł0

```
őł0 = őł0 - őĪ * (1/m) * ő£ (hőł(xi) - yi)
```

Update őł1

```
őł1 = őł1 - őĪ * (1/m) * ő£ (hőł(xi) - yi)xi
```

where

```
őĪ = learning rate
```

---

# Final Algorithm

```

őł0 = őł0 - őĪ * (1/m) * ő£ (hőł(xi) - yi)

őł1 = őł1 - őĪ * (1/m) * ő£ (hőł(xi) - yi)xi
```

---

# Key Insight

The gradient tells us:

```

```

Gradient descent simply **moves őł0 and őł1 in the direction that reduces the cost**.
