
# Simple Linear Regression

## Definition

Simple Linear Regression is used when:

* There is **one input feature**
* There is **one output feature**

The goal is to **predict the output value from the input value**.

Example

```
Input (X)  → House Size
Output (Y) → House Price
```

---

# Idea Behind Linear Regression

What the model tries to do:

* Plot **input values on the X-axis**
* Plot **output values on the Y-axis**
* Draw a **line that best represents the relationship between them**

This line is called the **Best Fit Line**.

---

# Why Do We Need a Best Fit Line?

Even if the relationship between variables is linear:

* The data points **do not lie perfectly on one line**
* There is **noise or variation in data**

Example visualization:

```
Data Points

Y
│        •
│    •
│          •
│  •
│      •
└────────────────── X
```

Because no single line passes through all points, we choose the line that:

* Stays **closest to all points**
* Makes **prediction error minimum**

This line is called the **Best Fit Line**.

---

# Representation of the Best Fit Line

A straight line has two components:

* Starting point on Y-axis (intercept)
* Change in Y with respect to X (slope)

Formula

```
y = mx + c
```

Where

```
y → predicted output
x → input value
m → slope of the line
c → intercept (value of y when x = 0)
```

Example

```
HousePrice = m * Size + c
```

---

# Prediction Using the Model

For every input `x`, the model predicts a value.

```
predicted_y = mx + c
```

But the **predicted value may not match the actual value**.

---

# Error in Prediction

Error is the difference between:

* Actual value
* Predicted value

Formula

```
error = actual_y - predicted_y
```

or

```
error = y - ŷ
```

Where

```
y  → actual value
ŷ  → predicted value
```

---

# Why Do We Square the Errors?

If we simply sum errors:

```
+5 + (-5) = 0
```

Positive and negative errors cancel out.

To avoid this, we **square the errors**.

```
Squared Error = (y - ŷ)^2
```

This ensures:

* All errors are positive
* Large errors are penalized more

---

# Cost Function

To measure how good a line is, we calculate **total error**.

This is called the **Cost Function**.

```
Cost = Sum of squared errors for all points
```

Formula

```
J(θ0, θ1) = (1 / 2m) * Σ (hθ(xi) - yi)^2
```

Where

```
θ0 → intercept
θ1 → slope
m  → number of data points
hθ(x) → predicted value
yi → actual value
```

Model equation

```
hθ(x) = θ0 + θ1x
```

---

# Understanding Candidate Lines

There are **many possible lines** that could fit the data.

Example:

```
Line 1 → slope = 0.5
Line 2 → slope = 1.2
Line 3 → slope = 2.0
```

Each line produces **different errors**.

So we calculate **cost for each line**.

The **best fit line is the one with minimum cost**.

---

# Cost Function Graph (When Intercept = 0)

If intercept is fixed at 0:

```
θ0 = 0
```

Only the **slope changes**.

If we plot:

```
X-axis → θ1 (slope)
Y-axis → J(θ1) (cost)
```

We get a **U-shaped curve**.

![Reference](https://editor.analyticsvidhya.com/uploads/97106gd4.jpeg)

The **lowest point of this curve** gives the **best slope**.

---

# Cost Function Graph (When Intercept ≠ 0)

If both parameters change:

```
θ0 → intercept
θ1 → slope
```

The cost graph becomes **3D**.

```
Shape: Bowl / Umbrella
```

Lowest point of this surface = **best parameters**.

---

# Finding the Best Parameters Efficiently

We cannot try thousands of slopes manually.

Instead we use an optimization algorithm called **Gradient Descent**.

---

# Gradient Descent (Convergence Algorithm)

Idea:

```
Repeat until convergence
```

Convergence means:

```
parameters stop changing significantly
```

Update rule

```
θ1 = θ1 - α * (dJ(θ1) / dθ1)
```

Where

```
θ1 → slope
α  → learning rate
dJ/dθ1 → slope of cost function
```

Explanation

```
If gradient is positive → decrease θ1
If gradient is negative → increase θ1
```

![Reference](https://editor.analyticsvidhya.com/uploads/48305gd2.png)

This keeps moving toward the **minimum cost**.

---

# Learning Rate

Learning rate controls **how big the update step is**.

```
α = learning rate
```

Typical value

```
0.001
```

Effects:

```
Small α → slow but stable learning
Large α → faster but may overshoot
```

---

# Gradient Descent with Both Parameters

When intercept is not zero, both parameters are updated.

```
θ0 = θ0 - α * (dJ(θ0, θ1) / dθ0)

θ1 = θ1 - α * (dJ(θ0, θ1) / dθ1)
```

These updates continue until:

```
cost function reaches minimum
```

---

# Final Goal

Find parameters:

```
θ0 (intercept)
θ1 (slope)
```

Such that:

```
Cost function J(θ0, θ1) is minimum
```

This gives us the **Best Fit Line**.
