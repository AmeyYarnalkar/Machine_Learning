

#  Supervised Learning

This is **the most common type of ML**.

Here the model learns from **labeled data**.

Meaning every input already has the **correct answer attached**.

Example dataset:

| House Size | Price   |
| ---------- | ------- |
| 1000 sq ft | 50 lakh |
| 1500 sq ft | 75 lakh |
| 2000 sq ft | 1 crore |

Here the model learns:

```
size → price
```

Then if you give a **new house size**, it predicts the price.

### Two main categories

**Regression**

Predicts a **continuous value**.

Examples:

* house price prediction
* stock price prediction
* temperature prediction

Algorithms:

* Linear Regression
* Ridge / Lasso
* Decision Trees
* Random Forest
* Gradient Boosting

---

**Classification**

Predicts **categories**.

Examples:

* spam or not spam
* disease or no disease
* cat or dog

Algorithms:

* Logistic Regression
* SVM
* Random Forest
* Naive Bayes
* Neural Networks

---

### Real world uses

* email spam detection
* credit scoring
* medical diagnosis
* recommendation systems

---

#  Unsupervised Learning

Here the data **has no labels**.

The model must **discover patterns on its own**.

Example dataset:

| Age | Salary |
| --- | ------ |
| 22  | 25k    |
| 24  | 27k    |
| 45  | 80k    |
| 48  | 85k    |

The algorithm might discover:

```
Young professionals
Senior professionals
```

### Main tasks

**Clustering**

Group similar data.

Algorithms:

* K-Means
* DBSCAN
* Hierarchical Clustering

Example uses:

* customer segmentation
* market analysis
* anomaly detection

---

**Dimensionality Reduction**

Reduce number of features while preserving information.

Algorithms:

* PCA
* t-SNE
* UMAP

Example:

Reducing **1000 features → 10 features**.

Useful for:

* visualization
* noise reduction
* faster training

---

### Real world uses

* fraud detection
* recommendation grouping
* social network analysis

---

#  Semi-Supervised Learning

This is a **hybrid** of supervised and unsupervised.

Data looks like this:

```
10,000 unlabeled images
500 labeled images
```

Labeling data is **expensive**, so we use small labeled data + lots of unlabeled data.

Example:

Medical imaging.

Doctors label a **few scans**, and the model learns from the rest.

Techniques:

* self-training
* pseudo labeling
* consistency regularization

---

#  Reinforcement Learning

This is completely different.

The model learns by **trial and error**.

Instead of labels, it gets **rewards or penalties**.

Example:

Training an AI to play a game.

Process:

```
Agent → takes action
Environment → gives reward
Agent → learns better strategy
```

Example:

Chess AI.

```
Win → +1 reward
Lose → -1 reward
```

Over time it learns optimal moves.

### Components

Agent
Environment
Action
State
Reward

---

### Algorithms

* Q-Learning
* Deep Q Network (DQN)
* Policy Gradient
* Actor-Critic
* PPO

(You earlier mentioned interest in **DQN**, which is exactly here.)

---

### Real world uses

* robotics
* game playing (AlphaGo)
* self-driving cars
* ad placement optimization
* dynamic pricing
