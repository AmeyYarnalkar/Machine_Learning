
# 1. Artificial Intelligence (AI)

**Artificial Intelligence = making machines behave intelligently.**

That’s it.

If a machine can **make decisions, solve problems, or mimic human intelligence**, it falls under AI.

Examples:

* Chess playing programs
* Self-driving cars
* Voice assistants
* Recommendation systems
* Spam filters

Important thing:

AI **does NOT require learning**.

Some AI systems use **rules written by humans**.

Example:

```
IF temperature > 40
    turn AC ON
```

This is **AI**, but **not Machine Learning**.

So:

**AI = the big umbrella.**

---

# 2. Machine Learning (ML)

Machine Learning is a **subset of AI**.

Instead of writing rules manually, we let the **machine learn patterns from data**.

Example problem:

Detect spam emails.

Old AI approach:

```
IF email contains "free money"
    mark spam
```

Machine Learning approach:

Feed **thousands of emails** to the model.

It learns patterns like:

* suspicious words
* sentence patterns
* sender behavior

Now it **predicts spam automatically**.

So:

**Machine Learning = AI that learns from data.**

---

# 3. Deep Learning (DL)

Deep Learning is **a subset of Machine Learning**.

It uses **Neural Networks with many layers**.

Inspired by the **human brain**.

Structure:

```
Input Layer
Hidden Layer
Hidden Layer
Hidden Layer
Output Layer
```

Why deep learning?

Because traditional ML struggles with **complex unstructured data** like:

* images
* audio
* video
* language

Deep learning handles these well.

Examples:

* Face recognition
* ChatGPT
* Self-driving cars
* Image generation
* Speech recognition

Popular architectures:

* CNN (images)
* RNN / LSTM (sequence data)
* Transformers (LLMs like GPT)

So:

```
AI
 └── ML
      └── Deep Learning
```

# 4. Data Science

Now here’s where people get confused.

**Data Science is not inside AI.**

It’s a **field that uses ML, statistics, and data analysis to extract insights from data.**

Goal:

**Understand data and make decisions.**

Example problems:

* Why are sales dropping?
* Which customers will churn?
* Which products will sell more next month?

Tools used:

* Statistics
* Machine Learning
* Data Visualization
* Data Cleaning
* Data Engineering

Workflow:

```
Collect data
Clean data
Analyze data
Build model
Interpret results
Communicate insights
```

Tools:

* Python
* Pandas
* NumPy
* SQL
* Matplotlib
* Scikit-learn

ML is just **one part** of Data Science.
