# Cost Function

## Definition

Cost function is used to:

```id="cf1"
Measure the overall error of a model
```

For a single data point:

```id="cf2"
error = actual value - predicted value
```

For the full dataset:

```id="cf3"
Cost = collective effect of all errors
```

---

# Types of Cost Functions

##  Mean Squared Error (MSE)

![Reference](https://rmitlibrary.github.io/cdn/learninglab/illustration/subject-support/maths-and-statistics/functions%20and%20graphs/quadratic%20functions_03@2x.png)

## Formula

```id="cf4"
MSE = (1/n) * Σ (yi - ŷi)^2
```

Where

```id="cf5"
yi  → actual value
ŷi  → predicted value
n   → number of data points
```

---

## Interpretation

```id="cf6"
If MSE = 2.3

→ On average, squared difference between
   actual and predicted values is 2.3
```

---

## Advantages

```id="cf7"
• Smooth curve (quadratic)
• Differentiable everywhere
• Has one global minimum
• Gradient descent works efficiently
```

---

## Disadvantages

```id="cf8"
• Very sensitive to outliers
• Large errors get squared → huge impact
• Output not in same unit as data
```

---

## Mean Absolute Error (MAE)

![Reference](https://s3-us-west-2.amazonaws.com/courses-images-archive-read-only/wp-content/uploads/sites/924/2015/11/25200923/CNX_Precalc_Figure_01_06_0032.jpg)

## Formula

```id="cf9"
MAE = (1/n) * Σ |yi - ŷi|
```

---

## Advantages

```id="cf10"
• Robust to outliers (no squaring)
• Same unit as original data
• Easy to interpret
```

---

## Disadvantages

```id="cf11"
• Not differentiable at zero
• Uses subgradients
• Optimization is slower
• Can have multiple local minima (harder convergence)
```

---

## Root Mean Squared Error (RMSE)

## Formula

```id="cf12"
RMSE = sqrt( (1/n) * Σ (yi - ŷi)^2 )
```

---

## Advantages

```id="cf13"
• Same unit as original data
• Differentiable
• Penalizes large errors (useful in many cases)
```

---

## Disadvantages

```id="cf14"
• Sensitive to outliers (due to squaring)
```

---

# Quick Comparison

| Metric | Outliers    | Differentiable | Units     | Convergence |
| ------ | ----------- | -------------- | --------- | ----------- |
| MSE    | ❌ sensitive | ✅ yes          | ❌ squared | ✅ fast      |
| MAE    | ✅ robust    | ❌ no (at 0)    | ✅ same    | ❌ slower    |
| RMSE   | ❌ sensitive | ✅ yes          | ✅ same    | ✅ fast      |

---

# Key Intuition

```id="cf15"
MSE → punishes large errors heavily

MAE → treats all errors equally

RMSE → like MSE but easier to interpret
```

---

# When to Use What

```id="cf16"
Use MSE → when you want strong penalty on large errors

Use MAE → when data has outliers

Use RMSE → when you want error in real units
           but still care about large errors
```
