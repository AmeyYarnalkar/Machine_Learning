# Silhouette Score 

## 1. Purpose of Silhouette Score

Silhouette score is used to **evaluate clustering quality**.

It measures two things:

1. **Cluster cohesion** → how close points are within the same cluster
2. **Cluster separation** → how far clusters are from each other

In simple terms:

> A good clustering means points are **close to their own cluster** and **far from other clusters**.

---

## 2. Two Important Quantities

For each data point (i), we compute two values.

---

## 2.1 Intra-cluster distance (a(i))

This measures **how close a point is to other points in the same cluster**.

It is the **average distance between point (i) and all other points in its cluster**.

Mathematically:

$$
a(i) =
\frac{1}{|C_i|-1}
\sum_{j \in C_i, j \neq i}
d(i,j)
$$

Where:

* $C_i$ = cluster containing point $i$
* $|C_i|$ = number of points in that cluster
* $d(i,j)$ = distance between points i and j

Meaning:

```text
a(i) = average distance from point i to all points in its own cluster
```

Interpretation:

* small (a(i)) → point fits well in cluster
* large (a(i)) → point poorly placed

---

## 2.2 Nearest-cluster distance (b(i))

Now we measure how far the point is from the **next best cluster**.

Steps:

1. Compute the average distance from point (i) to **all points in another cluster**
2. Do this for **every other cluster**
3. Choose the **minimum of those distances**

Mathematically:

$$
b(i) =
\min_{C_k \neq C_i}
\left(
\frac{1}{|C_k|}
\sum_{j \in C_k}
d(i,j)
\right)
$$

Meaning:

```text
b(i) = average distance from point i to the nearest neighboring cluster
```

Interpretation:

* small $b(i)$ → cluster overlap
* large $b(i)$ → clusters well separated

---

## 3. Silhouette Score Formula

For each point (i):

$$
s(i) =
\frac{b(i) - a(i)}
{\max(a(i), b(i))}
$$

---

## 4. Score Range

$$
-1 \leq s(i) \leq 1
$$

---

## 5. Interpretation

| Score    | Meaning                                |
| -------- | -------------------------------------- |
| **≈ 1**  | point is well clustered                |
| **≈ 0**  | point lies between clusters            |
| **≈ -1** | point likely assigned to wrong cluster |

---


## 6. Using Silhouette Score to Choose K

Procedure:

1. Run clustering for different values of (K)

```text
K = 2,3,4,5...
```

2. Compute silhouette score for each
3. Choose the **K with highest score**

Example:

| K | Score |
| - | ----- |
| 2 | 0.41  |
| 3 | 0.58  |
| 4 | 0.52  |

Best choice:

```text
K = 3
```

---

## 8. Intuition

Silhouette score balances two goals:

```text
minimize intra-cluster distance
maximize inter-cluster distance
```

Which means:

```text
points close to their cluster
clusters far from each other
```
